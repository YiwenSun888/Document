Welcome to HME_DataAnalyzer_Yiwen’s documentation!¶

HME_DataAnalyzer_Yiwen is a fast parser and editor for ASAM (Association for Standardization of Automation and Measuring Systems) MDF (Measurement Data Format) files.

HME_DataAnalyzer_Yiwen supports MDF versions 2 (.dat), 3 (.mdf) and 4 (.mf4).

HME_DataAnalyzer_Yiwen works on Python >= 3.7 (for Python 2.7, 3.4 and 3.5 see the 4.x.y releases)
